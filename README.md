## なぜこのアプリを作ったの？
このアプリを作ろうと思ったきっかけはよく一緒に外食に行く友人が、以前撮った料理の写真を見ながら  
「写真撮るのは良いけどどこのお店か忘れてるものもある」と言った一言でした。  
プログラミングを学んでいた私は、お店の情報と画像を保存できるウェブサイトを作ってしまえば良いと考え、
アプリ作成に至りました。<br>
この時、作ってしまえば良い、と自然と考えられたことに気づいた時、
プログラミングの可能性を感じました。
